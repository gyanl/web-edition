Line Height Scale

Line heights tighten as text size increases. Large headings use 1.1, smaller headings use up to 1.5, and body text uses 1.5 for readability.

Element Line Height Responsive Notes
H1 1.2 → 1.1 Mobile → Desktop Tightest line height for hero headings
H2 1.2 → 1.1 Mobile → Desktop Same as H1
H3 1.3 — Slightly looser for section headings
H4 1.4 — Transitional line height
H5 1.4 — Same as H4
H6 1.5 — Matches body text line height
Body / p / li 1.5 — Comfortable reading line height
Global (body) 140% — Base body line height (overridden by elements above)
